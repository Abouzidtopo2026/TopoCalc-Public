# TopoCalc - Version Publique

Cet outil est une application de calcul topographique (Progressive Web App et Android) développée pour assister les géomètres, topographes et projeteurs géomètres dans la conception, le calcul et l'implantation géométrique.

**Ce dépôt est destiné à la distribution publique et contient uniquement les ressources pour installer et apprendre à utiliser l'application.**

## 📦 Contenu
- `TOPO_CALC_Release_V01.apk` : L'application Android compilée, prête à être installée sur votre smartphone.
- `MANUEL_UTILISATEUR.pdf` / `MANUEL_UTILISATEUR.html` : La documentation détaillée expliquant le modèle mathématique des calculs (Axe en Plan et Profil en Long) et le manuel d'utilisation de l'interface.
- `donnees_exemple/` : Exemples de fichiers pour l'application :
  - Sauvegardes de projets `.json`
  - Matrices `.xlsx`
  - Rapports d'implantation `.csv`

## 🛠️ Installation Android
1. Téléchargez le fichier `TOPO_CALC_Release_V01.apk` depuis ce dépôt.
2. Ouvrez-le sur votre appareil Android.
3. Autorisez l'installation d'applications issues de "sources inconnues" si demandé.
4. Lancez TopoCalc !

## 🍏 Utilisation sur iPhone / iPad (iOS)
Grâce au format Web App (PWA), TopoCalc est directement utilisable et installable sur les appareils Apple.

1. Ouvrez **Safari** sur votre iPhone ou iPad.
2. Allez sur le lien web officiel de l'application : **[https://topocalc.netlify.app/](https://topocalc.netlify.app/)**
3. Touchez l'icône de **Partage** en bas de l'écran (le petit carré avec une flèche pointant vers le haut).
4. Faites défiler le menu et sélectionnez **"Sur l'écran d'accueil"** (Add to Home Screen).
5. Appuyez sur **Ajouter** en haut à droite.

L'icône TopoCalc apparaîtra alors parmi vos applications et s'ouvrira en plein écran, sans la barre du navigateur !
